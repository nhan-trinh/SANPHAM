# Login Page GUI Testing (Playwright)

## Mô tả
Dự án này là demo kiểm thử giao diện đăng nhập sử dụng Playwright.

## Cài đặt
```bash
npm install
npx playwright install
```

## Chạy frontend
Dùng Live Server hoặc Express server để chạy `index.html` tại `http://localhost:3000`

## Chạy test
```bash
npx playwright test
```

## Xem báo cáo
```bash
npx playwright show-report
